# Melhorias Futuras — Gerador de Currículo ATS

## 1. Objetivo

Este documento lista possíveis melhorias para evoluir o projeto Gerador de Currículo ATS.

A versão inicial é simples e focada em análise de compatibilidade entre currículo e vaga.

## 2. Integração com IA

Adicionar integração com IA para gerar recomendações mais completas.

Possibilidades:

- Melhorar resumo profissional.
- Reescrever experiências.
- Sugerir habilidades técnicas.
- Criar currículo otimizado.
- Gerar carta de apresentação.

## 3. Exportação em PDF

Permitir que o usuário baixe o currículo otimizado em PDF.

Essa funcionalidade tornaria o projeto mais útil para uso real.

## 4. Exportação em DOCX

Permitir exportação em Word para que o usuário possa editar o currículo depois.

## 5. Upload de Currículo

Adicionar upload de arquivos em:

- PDF
- DOCX
- TXT

Assim o usuário não precisaria copiar e colar o currículo manualmente.

## 6. Histórico de Análises

Salvar análises anteriores para comparar a evolução do currículo.

Dados que poderiam ser salvos:

- Nome da vaga
- Data da análise
- Score ATS
- Palavras-chave ausentes
- Recomendações

## 7. Login de Usuário

Adicionar login para cada usuário salvar seus currículos e análises.

Uma opção futura seria usar Supabase.

## 8. Banco de Dados

Criar banco de dados para armazenar:

- Usuários
- Currículos
- Vagas analisadas
- Scores
- Recomendações

## 9. Dashboard de Evolução

Criar um painel mostrando:

- Melhor score obtido
- Média de score
- Quantidade de vagas analisadas
- Palavras-chave mais ausentes
- Evolução do currículo

## 10. Comparação com Múltiplas Vagas

Permitir comparar o mesmo currículo com várias vagas ao mesmo tempo.

Isso ajudaria o candidato a escolher quais vagas têm mais aderência ao perfil.

## 11. Versão em Inglês

Criar suporte para currículo em inglês.

Isso seria útil para vagas remotas e internacionais.

## 12. Melhorias de Interface

Possíveis melhorias visuais:

- Tela de prévia do currículo.
- Botão para copiar resultado.
- Barra de score com cores.
- Modo escuro.
- Layout mobile melhorado.

## 13. Segurança e Privacidade

Como o currículo contém dados pessoais, uma versão futura deve considerar:

- Não expor dados sensíveis.
- Evitar salvar informações sem consentimento.
- Proteger chaves de API.
- Não enviar `.env` para o GitHub.

## 14. Roadmap

- Banco de dados

## 15. Conclusão

O projeto pode começar simples, mas tem potencial para evoluir bastante.

Ele resolve um problema real de candidatos que precisam adaptar seus currículos para vagas e sistemas ATS.

Também é um bom projeto de portfólio porque combina desenvolvimento web, carreira, análise de texto, IA e experiência do usuário.
