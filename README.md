# Gerador de Currículo ATS

Projeto web criado para analisar a compatibilidade entre um currículo e uma descrição de vaga, simulando uma análise ATS.

## Objetivo

O objetivo do projeto é ajudar candidatos a identificarem se o currículo está alinhado com uma vaga específica.

A aplicação permite inserir dados do currículo, colar a descrição da vaga e visualizar uma análise com score, palavras-chave encontradas, palavras-chave ausentes e recomendações de melhoria.

## Funcionalidades

- Inserção de dados do candidato
- Campo para descrição da vaga
- Análise de compatibilidade ATS
- Score de 0 a 100
- Palavras-chave encontradas
- Palavras-chave ausentes
- Recomendações de melhoria
- Interface simples e responsiva

## Fluxo Técnico

Usuário acessa a aplicação
        ↓
Preenche os dados do currículo
        ↓
Cola a descrição da vaga
        ↓
Clica em analisar
        ↓
O sistema compara currículo e vaga
        ↓
O sistema calcula o score ATS
        ↓
O resultado é exibido na tela
